# music-web (static)
Demo generador musical WebAudio con créditos y planes (sin build).

## Estructura
- `index.html` — aplicación completa (React + Tailwind por CDN)
- `.github/workflows/pages.yml` — despliegue automático a GitHub Pages
- `.nojekyll` — evita procesamiento Jekyll

## Publicación
1. Sube estos archivos al repo.
2. En *Settings → Pages*, selecciona **Source: GitHub Actions**.
3. Revisa la pestaña **Actions** y espera a que termine el workflow.
4. Tu sitio quedará disponible en ` https://ivanplaysyt.github.io/music-web/`.
